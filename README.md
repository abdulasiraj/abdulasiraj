## About

Senior Data Engineer with 5+ years designing and shipping production-grade data platforms across finance, spend analytics, and machine-learning domains. I focus on the layer where **modelling, infrastructure, and delivery meet** — dbt on the warehouse, Terraform underneath, and CI/CD wired together so every change is safe, reviewed, and reversible.

Currently at **[Telenor Shared Services](https://tss.telenor.com/)**, where I architected the TSS Data Platform on Google Cloud — a multi-domain lakehouse serving 15+ markets.

## What I do

- **Data platforms & modelling** — dbt on BigQuery / Snowflake / Databricks, bronze / silver / gold layer contracts, semantic models, and data contracts.
- **Infrastructure as code** — Terraform for cloud resources, IAM, and access management, with plan-on-PR / apply-on-merge CI.
- **Keyless CI/CD** — GitHub Actions with Workload Identity Federation, containerised delivery, and auditable releases.
- **Applied GenAI on the warehouse** — vector search and embeddings for semantic matching and classification, right next to the data.
- **Reliability & quality** — automated testing, source freshness, observability, and documentation baked into the pipeline.
